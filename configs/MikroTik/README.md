Mikrotik est une entreprise d'équipement réseau basé en Lettonie. Pourquoi ce choix ?

- Excellent rapport qualité-prix
- Une communauté active et un support fiable
- Idéal pour les fournisseurs de services Internet

J'utilise le modèle RB3011UiAS-RM mais il est possible d'avoir un Mikrotik dans une VM : RouterOS (Le système d'exploitation des produits de la marque)
Vu que je ne dépasse pas 1Gpbs et que je voulais habiller ma magnifique baie 9U, j'ai acheté ce modèle pour 50€ en Belgique.
Dans ce dossier, je vais vous apprendre les commandes à connaître, comment j'ai configuré mon tunnel IP et certaines notions de réseau à avoir pour la suite.
